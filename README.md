# Inventory Management & Demand Forecasting System

## Project Overview
This project is a Python-based Inventory Management System that helps track stock levels, manage inventory operations, and forecast future demand using historical sales data. It also includes features like reorder alerts, inventory valuation, and Economic Order Quantity (EOQ) calculation.

---

## Features

- View current inventory details
- Add or sell stock items
- Automatic low-stock reorder alerts
- Inventory value calculation
- Demand forecasting using past sales (moving average method)
- Economic Order Quantity calculation

---

## Technologies Used

- Python
- Pandas
- CSV file handling
